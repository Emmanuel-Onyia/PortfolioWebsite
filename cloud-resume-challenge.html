Cloud Challenge

This is a project I created as part of the Cloud Resume Challenge. The challenge includes building a cloud-hosted resume/portfolio and setting up infrastructure as code (IaC) using AWS.

Project Details

  • Created and customised the site locally using html, css and javascript
  • Configured an S3 bucket to host the resume files.
  • Secured the resume using CloudFront and Route 53 for domain management.
  • Used AWS Certificate Manager to provision and manage SSL/TLS certificates with AWS services.

The project highlights my skills in html/css, AWS cloud sevices and security.

The project will be updated regularly with more learning
